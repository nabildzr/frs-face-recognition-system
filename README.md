# Face Recognition

This project is being developed by [@putra2078](https://github.com/putra2078) & [@nabildzr](https://github.com/nabildzr).

## Setup

### 1. Clone repository

```bash
git clone https://github.com/nabildzr/frs-face-recognition-system
cd face_recognition
```

### 2. Create Virtual Environment

```bash
python -m venv your_virtual_environment
```

### 3. Active the Virtual Environment

**Windows (Command Prompt):**

```bash
your_virtual_environment\Scripts\activate
```

**Windows (PowerShell):**

```bash
your_virtual_environment\Scripts\Activate.ps1
```

**Linux/Mac:**

```bash
source your_virtual_environment/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

## Run

```
 source your_virtual_environment/Scripts/activate && python frec.py
```
